## 驱动一路

| 引脚 | 描述               |
| ---- | ------------------ |
| PWMA | 电机1PWM           |
| AIN1 | 电机1转向控制1     |
| AIN2 | 电机1转向控制2     |
| AO1  | 电机1正极          |
| AO2  | 电机1负极          |
| BMQ1 | 电机1编码器输出A相 |
| BMQ2 | 电机1编码器输出B相 |

##  驱动二路

| 引脚 | 功能               |
| ---- | ------------------ |
| PWMB | 电机2PWM           |
| BIN1 | 电机2转向控制1     |
| BIN2 | 电机2转向控制2     |
| BO1  | 电机2正极          |
| BO2  | 电机2负极          |
| BMQ3 | 电机2编码器输出A相 |
| BMQ4 | 电机2编码器输出B相 |
